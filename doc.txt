у параграфов класс plain-text
у сайдбара класс sidenav
у хедбара класс headnav
у ссылок на якори класс navigation
у изображений класс image
у подписей изображений на текстовой версии класс plain-text-img
у ссылок класс link

Касательно структуры проекта:
  $ROOT/
    pages/страница.txt 
         /изображение.png
         /1/страница.txt
          1/1.png
  превращается в
  $OUTPUT/
    $mode/$color/index.html 
                /1/index.html
    $images/image.png
           /1/1.png
В любой поддиректории $ROOT должен быть лишь одна страница page.txt
$mode - может быть desktop, mobile или plain-text
$color - может быть dark или light
Все ссылки на изображения заменяются.

Синтаксис файла. Теги.
(корень) - тег, обозначающий корень
(заголовок) - заголовок страницы
(блок идентификатор) - блок(идентификатор идёт в боковую панель)
(изображение подпись) - изображение(если оно не прогрузилось или версия сайта plain-text, то отображается подпись) 
(параграф) - туда писать текст
Все теги необходимо закрыть.

Вид сайта:
сверху панель для перемещения по сайту
с правого бока сайта панель для перемещения по блокам в странице

Древовидный вид сайта упростит навигацию по нему.
